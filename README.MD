ACCOUNT BANK

Projeto desenvolvido em Java que simula operações básicas de uma conta bancária, como depósito, saque e consulta de saldo.

FUNCIONALIDADES

- Criar contas bancárias
- Realizar depósitos
- Realizar saques
- Consultar saldo atual

TECNOLOGIAS USADAS

- Java
- [Eclipse](https://www.eclipse.org/) (IDE)
